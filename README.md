# Проект: Анализ банковских транзакций

## Описание

Данный проект содержит набор утилит на языке Python для обработки и анализа списка банковских операций. Инструменты позволяют фильтровать транзакции по статусу выполнения и сортировать их по дате.

## Установка

Для работы с функциями не требуется установка дополнительных библиотек. Достаточно иметь интерпретатор Python версии 3.14 или выше.

1. Скачай файл с кодом (например, utils.py) или вставь функции в свой скрипт.
2. Создай файл main.py (или используй интерактивную оболочку Python).
3. Импортируй функции и используй их согласно примерам ниже.

## Описание функций

`
filter_by_state(operations, state='EXECUTED')
`

Функция принимает список словарей с банковскими операциями и возвращает новый список, содержащий только те словари, у которых ключ state соответствует указанному значению.

#### Аргументы:

* operations (list[dict]): Список словарей с данными об операциях.
* state (str, optional): Значение статуса для фильтрации. По умолчанию 'EXECUTED'.

#### Возвращает:

* list[dict]: Новый список словарей, прошедших фильтрацию.

`
sort_by_date(data, reverse=True)
`

#### Аргументы:

* data (list[dict]): Список словарей с данными об операциях.
* reverse (bool, optional): Порядок сортировки. По умолчанию True.

#### Возвращает:

* list[dict]: Новый отсортированный список словарей.

## Примеры использования

### Подготовка данных

#### Исходный список транзакций

`
data = [    {"id": 1, "date": "2023-09-01T12:00:00", "state": "EXECUTED", "amount": 5000},    {"id": 2, "date": "2023-10-15T14:30:00", "state": "CANCELED", "amount": 3000},    {"id": 3, "date": "2023-08-20T09:15:00", "state": "EXECUTED", "amount": 10000},    {"id": 4, "date": "2023-11-05T18:45:00", "state": "EXECUTED", "amount": 2000},]
`

1. Фильтрация транзакций

Получим только успешно выполненные операции (EXECUTED).

```
Вызываем функцию фильтрации
executed_only = filter_by_state(data)
```

```
Выводим результат
print(executed_only)
```

#### Результат:

```
[
    {'id': 1, 'date': '2023-09-01T12:00:00', 'state': 'EXECUTED', 'amount': 5000},
    {'id': 3, 'date': '2023-08-20T09:15:00', 'state': 'EXECUTED', 'amount': 10000},
    {'id': 4, 'date': '2023-11-05T18:45:00', 'state': 'EXECUTED', 'amount': 2000}
]
```

2. Сортировка по дате (по убыванию)

Отсортируем выполненные операции так, чтобы самые новые были первыми.

```
# Вызываем функцию сортировки (по умолчанию reverse=True)
sorted_desc = sort_by_date(executed_only)

print(sorted_desc)
```

#### Результат:

```
[
    {'id': 4, 'date': '2023-11-05T18:45:00', 'state': 'EXECUTED', 'amount': 2000}, # Ноябрь (самая новая)
    {'id': 1, 'date': '2023-09-01T12:00:00', 'state': 'EXECUTED', 'amount': 5000}, # Сентябрь
    {'id': 3, 'date': '2023-08-20T09:15:00', 'state': 'EXECUTED', 'amount': 10000} # Август
]
```
